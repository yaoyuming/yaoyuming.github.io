JDK 安装（Windows）

JDK 安装

一、下载地址

  • 官网地址：https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html

  • 华为提供的下载服务：https://repo.huaweicloud.com/java/jdk/

  • 清华大学提供的下载服务：https://mirrors.tuna.tsinghua.edu.cn/AdoptOpenJDK/

二、环境变量配置

新建 JAVA_HOME

添加 CLASSPATH

变量名：CLASSPATH
变量值：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar

更改 Path

点击编辑，在弹出的新建对话框中点击新建，添加%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin。

测试 JDK

  1. java
  2. java -version
  3. javac
